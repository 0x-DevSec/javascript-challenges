# Challenge Grid – Instructions & Descriptions

## Structure du Projet

### 1. HTML
Dans un fichier **HTML**, créez **40 divs** dans un container.  
Le container parent doit utiliser **display: grid** pour organiser la mise en page.

Chaque div doit avoir un **ID unique**, par exemple :

- `#challenge-1`
- `#challenge-2`
- `#challenge-3`
- ...
- `#challenge-40`

> Note : Il y a **moins que 40 challenges**, car certains ont été supprimés.  
> Je vais utiliser ce fichier comme base pour la mise en situation du débriefing (lundi après-midi).  
> Bon courage !

### 2. JavaScript
Dans un fichier **JS**, vous devez traiter chaque challenge.  
Utilisez des **commentaires clairs** pour séparer et expliquer chaque fonctionnalité.
